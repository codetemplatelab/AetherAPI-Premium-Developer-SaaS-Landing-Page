# AetherAPI - Premium Developer API Platform Template

AetherAPI is a high-end, production-ready SaaS landing page template designed specifically for developer-focused platforms. Built with React, TypeScript, Tailwind CSS, and Framer Motion, it features a modern "Cyber Emerald" dark theme, interactive 3D elements, and a scalable architecture.

## Features

- **Developer-First UI:** Clean, technical aesthetic inspired by Stripe and Vercel.
- **Interactive 3D Hero:** Lightweight Three.js animation using React Three Fiber.
- **Multi-Protocol Support:** Detailed setup guides for REST, GraphQL, and Socket.io.
- **Native SDKs:** Code examples for JavaScript, Python, and Go.
- **Interactive Setup Flow:** Simulated project creation and deployment wizard.
- **Real-time Analytics:** Dashboard preview with live-updating charts.
- **Highly Animated:** Smooth scroll reveals, hover effects, and micro-interactions powered by Framer Motion and GSAP.
- **Fully Responsive:** Optimized for mobile, tablet, and desktop.
- **Modern Tech Stack:** React 19, Vite, TypeScript, Tailwind CSS 4, Material UI.
- **Dark Mode First:** Premium dark theme with optional light mode support.

## Tech Stack

- **Framework:** React.js (Vite)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Components:** Material UI (MUI)
- **Icons:** Lucide React
- **Animations:** Framer Motion
- **3D Graphics:** Three.js / React Three Fiber
- **Charts:** Recharts

## Getting Started

### Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

Start the development server:
```bash
npm run dev
```

### Building for Production

Build the project:
```bash
npm run build
```

The output will be in the `dist/` folder, ready for deployment to Vercel, Netlify, or any static hosting provider.

## Folder Structure

- `src/components/layout`: Global layout components like Navbar and Footer.
- `src/components/sections`: Individual landing page sections.
- `src/components/ui`: Reusable UI primitives (Buttons, Cards, etc.).
- `src/components/animations`: Animation wrappers and 3D components.
- `src/data`: Dummy content and configuration.
- `src/hooks`: Custom React hooks.
- `src/styles`: Global styles and Tailwind configuration.
